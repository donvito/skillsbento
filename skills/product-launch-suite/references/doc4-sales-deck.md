# Document 4: Sales Deck & Case Studies (.pptx)

## Presentation Structure

This is a visual sales deck in PowerPoint format. It should be 12-16 slides with bold, engaging design — NOT boring corporate bullets. Every slide must have a visual element.

**Read `/mnt/skills/public/pptx/SKILL.md` and `/mnt/skills/public/pptx/pptxgenjs.md` before creating this document.**

## Slide Breakdown

### Slide 1: Title Slide
- **Layout**: Dark background, large product name, tagline
- **Content**:
  - Product name (large, bold)
  - One-line tagline / value proposition
  - Company logo placeholder
  - Date / presenter name
- **Design**: Full dark background (e.g., #1E2761 navy or #2C2C2C dark gray), white text, clean and impactful

### Slide 2: The Problem
- **Layout**: Icon + text rows, or large stat callouts
- **Content**:
  - 3-4 pain points the target customer faces
  - Include a compelling statistic from research (e.g., "73% of teams report X")
  - Make it emotional — describe the frustration, not just the fact
- **Design**: Use icons or visual indicators next to each pain point. Consider a dark or accent-colored background to set a serious tone.

### Slide 3: The Solution
- **Layout**: Two-column (text left, visual/mockup right) or centered statement
- **Content**:
  - Clear, one-sentence solution statement
  - 3-4 key capabilities that address the problems
  - How it's different from alternatives
- **Design**: Clean, bright background. The solution should feel like relief after the problem slide.

### Slide 4-5: How It Works (Demo / Walkthrough)
- **Layout**: Step-by-step process flow or numbered sequence
- **Content**:
  - 3-5 steps showing the user journey
  - Step 1: [Action] → Step 2: [Action] → Step 3: [Result]
  - Each step with a brief description
- **Design**: Numbered circles or arrows showing flow. Use a light background with accent colors for step indicators.

### Slide 6: Key Features & Benefits
- **Layout**: 2x2 or 2x3 grid of feature cards
- **Content**:
  - 4-6 key features, each with:
    - Feature name (bold)
    - One-line benefit description
    - Icon or visual indicator
- **Design**: Grid of colored cards or icon+text blocks. Each feature gets its own visual "card" with consistent styling.

### Slide 7: Market Opportunity
- **Layout**: Large stat callouts with supporting text
- **Content**:
  - TAM / SAM / SOM from Document 1 (simplified)
  - 2-3 large numbers (e.g., "$4.2B market", "32% CAGR", "10M+ potential users")
  - Brief context for each number
- **Design**: Big, bold numbers (60-72pt) with smaller labels. Use the accent color for the numbers.

### Slide 8: Competitive Advantage
- **Layout**: Comparison table or positioning visual
- **Content**:
  - Simplified competitive comparison (us vs. top 2-3 competitors)
  - Focus on key differentiators
  - Use checkmarks / X marks for feature comparison
  - Or describe competitive positioning quadrant
- **Design**: Clean table with our column highlighted. Or a visual positioning map description.

### Slide 9-10: Case Studies / Success Stories
- **Layout**: Quote + metrics format, or story arc (before → after)
- **Content per case study**:
  - Company name (or "[Industry] company" if hypothetical)
  - Challenge they faced
  - Solution applied
  - Results achieved (with specific metrics)
  - Pull quote from customer
- **Design**: Large quote marks for testimonials. Stat callouts for results (e.g., "47% reduction in X"). Use a subtle accent background.
- **Note**: If no real case studies exist yet, create hypothetical scenarios based on the target ICP and expected outcomes. Clearly mark as "Projected Scenario" or "Expected Outcome".

### Slide 11: Social Proof & Testimonials
- **Layout**: Grid of logos + testimonial quotes
- **Content**:
  - Customer or partner logos (or placeholder boxes)
  - 2-3 short testimonial quotes
  - Trust indicators: certifications, awards, press mentions
  - Key metrics: "X customers", "Y% satisfaction", "Z uptime"
- **Design**: Logo grid at top, quotes below. Clean, trustworthy feel.
- **[PLACEHOLDER: Insert actual customer logos and testimonials]**

### Slide 12: Pricing Overview
- **Layout**: 3-column pricing comparison
- **Content**:
  - 3 pricing tiers (from Document 1)
  - Tier name, price, key features per tier
  - "Most Popular" callout on recommended tier
  - "Contact us for Enterprise" or custom pricing note
- **Design**: Three equal-width columns with the middle/recommended tier visually highlighted (accent color header or border).

### Slide 13: Call to Action / Next Steps
- **Layout**: Centered, bold CTA with supporting details
- **Content**:
  - Clear primary CTA: "Start your free trial", "Schedule a demo", "Contact us"
  - Supporting next steps (2-3 items)
  - Contact information: email, website, phone
  - QR code placeholder for quick access
- **Design**: Dark background matching the title slide. Large CTA text. Create visual bookend with Slide 1.

### Slide 14 (Optional): Appendix / Technical Details
- **Layout**: Clean text with structured information
- **Content**:
  - Technical specifications
  - Security and compliance details
  - Integration capabilities
  - SLA details
- **Design**: Simple, informational. This slide supports due diligence, not the pitch.

## Design Guidelines

### Color Palette Selection
Choose a palette that fits the product's industry and brand:

| Industry | Suggested Palette |
|----------|------------------|
| FinTech / Enterprise | Midnight Executive: `1E2761` / `CADCFC` / `FFFFFF` |
| Health / Wellness | Sage Calm: `84B59F` / `69A297` / `50808E` |
| Developer Tools | Charcoal Minimal: `36454F` / `F2F2F2` / `212121` |
| Consumer / Social | Coral Energy: `F96167` / `F9E795` / `2F3C7E` |
| Real Estate / PropTech | Warm Terracotta: `B85042` / `E7E8D1` / `A7BEAE` |
| AI / Technology | Ocean Gradient: `065A82` / `1C7293` / `21295C` |
| Sustainability | Forest & Moss: `2C5F2D` / `97BC62` / `F5F5F5` |

### Typography
- **Header font**: Arial Black, Trebuchet MS, or Georgia (pick one for the whole deck)
- **Body font**: Calibri or Arial
- **Title size**: 36-44pt bold
- **Body size**: 14-16pt
- **Stat callouts**: 48-72pt bold

### Slide Design Principles
- **Dark bookend slides**: Title and CTA slides should have dark backgrounds
- **Light content slides**: Body slides should have light backgrounds for readability
- **One visual per slide minimum**: Chart, icon grid, stat callout, comparison table, or image placeholder
- **Breathing room**: 0.5" minimum margins, don't crowd content
- **Consistent spacing**: Use the same gaps between elements across all slides
- **NO accent lines under titles** — use whitespace or background color contrast instead

### Shapes and Visual Elements
- Use rounded rectangles for feature cards (cornerRadius: 0.1)
- Use circles for step numbers or icons
- Use subtle gradient or solid color backgrounds for emphasis slides
- Ensure all text has strong contrast against its background

## Content Quality for Sales Decks

- **Lead with impact**: Every slide should make the viewer want to see the next one
- **One idea per slide**: Don't overload — split into multiple slides if needed
- **Numbers sell**: Use specific data points from research (market size, growth rates, ROI metrics)
- **Customer-centric language**: "You" and "Your" over "We" and "Our"
- **Clear hierarchy**: Most important info should be the largest / most prominent
- **End with urgency**: The CTA should motivate immediate action
